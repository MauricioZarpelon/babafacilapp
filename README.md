# Babá Fácil 👶📱

Aplicativo Android desenvolvido para facilitar a conexão entre pais/responsáveis e babás, oferecendo uma forma simples e organizada de cadastrar e encontrar profissionais para o cuidado infantil.

## 🎯 Objetivo
O Babá Fácil nasceu como um projeto prático de aprendizado em Android moderno, com foco em resolver um problema real do dia a dia, unindo funcionalidade, simplicidade e boa experiência do usuário.

## 🛠️ Tecnologias Utilizadas
- Kotlin  
- Android Studio  
- Jetpack Compose  
- Firebase (Firestore e Storage)

## ✨ Funcionalidades Atuais
- Cadastro de babás com informações básicas  
- Upload de foto do perfil  
- Busca de babás cadastradas  

## 🚧 Status do Projeto
Projeto em desenvolvimento inicial, com estrutura funcional e espaço para evolução.

## 📦 Publicação
Disponível em **teste fechado** na Google Play.
